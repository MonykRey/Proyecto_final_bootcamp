  • Grupo de aves White Rock: 5 machos apareados con 3 hembras cada uno

  • Male: factor con 5 niveles
  • Female: factor con 15 niveles

4. Preguntas de análisis

a. Variación dentro de una misma cruza

  • Desviación estándar promedio dentro de cruza: ~74 g
  • Cruzas más consistentes: M5 x F13 (SD ~26.9 g), M5 x F14 (SD ~28.3 g)
  • Cruzas más dispersas: M1 x F01 (SD ~104 g), M2 x F05 (SD ~107 g)
  • La variación no es homogénea entre familias: posibles causas → variabilidad individual, incubación, orden de eclosión

b. ¿Genética o ambiente?

Se ajustó un modelo mixto (nlme::lme) con macho y hembra anidada como efectos aleatorios.

  • 32% de la variación → varianza genética aditiva (medios hermanos paternos)
  • 68% de la variación → varianza ambiental (efecto materno + residual)
  • Contribución genética relevante, pero la mayoría de la variación es no genética
Code 
model <- lme(weight ~ 1, random = ~ 1 | male/female, data = D)
vc <- VarCorr(model)
var_macho    <- as.numeric(vc[2, 1])
var_hembra   <- as.numeric(vc[4, 1])
var_residual <- as.numeric(vc[5, 1])
var_genetica  <- 4 * var_macho
var_ambiental <- var_residual + var_hembra

var_df <- data.frame(
  fuente = c("Genética", "Ambiental"),
  varianza = c(var_genetica, var_ambiental)
) %>%
  mutate(total = sum(varianza), porcentaje = round(varianza / total * 100, 1))

Gráfica: Partición de varianza

d. Predicción de cruzas nuevas

Predicción = Intercepto (808.47 g) + Efecto genético del macho (BLUP)

  • Nueva cruza con M5 (mejor reproductor) → 829.64 g esperados
  • Nueva cruza con M1 (menor mérito) → 785.16 g esperados
  • Limitación: el efecto de la hembra está anidado dentro de su macho original, por lo que no es transferible a otras cruzas
Code 
machos  <- unique(D$male)
hembras <- unique(D$female)
intercepto <- as.numeric(fixef(model)[1])

predicciones_nuevas <- expand.grid(male = machos, female_nueva = hembras) %>%
  left_join(ranef_macho %>% select(male, efecto_macho = efecto), by = "male") %>%
  mutate(peso_predicho = intercepto + efecto_macho) %>%
  anti_join(D %>% select(male, female_nueva = female), by = c("male", "female_nueva")) %>%
  arrange(desc(peso_predicho))

pred_resumen <- predicciones_nuevas %>%
  distinct(male, peso_predicho) %>%
  arrange(desc(peso_predicho))

Gráfica: Peso predicho por macho

e. Consistencia del efecto del macho

¿El efecto de un macho es estable sin importar la hembra, o depende de la combinación específica?

  • M3: muy consistente (SD = 8.9 g) → efecto genético estable
  • M1: muy inconsistente (SD = 80.1 g), varía de 700 g a 848 g según la hembra
  • M4 y M5: variación moderada (SD = 53.3 g y 70.7 g)
Code 
medias_cruza <- D %>%
  group_by(male, female) %>%
  summarise(media_cruza = mean(weight), .groups = "drop")

consistencia_macho <- medias_cruza %>%
  group_by(male) %>%
  summarise(
    media_general = mean(media_cruza),
    sd_entre_hembras = sd(media_cruza),
    rango = max(media_cruza) - min(media_cruza),
    cv_pct = round(sd_entre_hembras / media_general * 100, 1)
  ) %>%
  arrange(sd_entre_hembras)

Gráfica: Consistencia entre machos

5. Conclusiones

Hallazgos principales

  • El peso de la progenie varía en promedio ~74 g dentro de una misma cruza, con notable heterogeneidad entre familias (SD entre ~27 g y ~107 g)
  • 68% de la variación es de origen ambiental; solo 32% es varianza genética
  • M5 es el mejor reproductor (+21.2 g); M1 el de menor mérito (-23.3 g)
  • Entre hembras, F14 y F01 destacan; F03 y F02 tienen el menor aporte

Implicaciones prácticas

  • El modelo predice con confianza el peso esperado de una cruza nueva a partir del efecto del macho
  • No es posible anticipar el efecto de una hembra fuera de su cruza original (diseño anidado hembra dentro de macho)
  • El efecto del macho no siempre es consistente: M3 es estable (SD = 8.9 g), M1 depende fuertemente de la hembra (SD = 80.1 g)
  • Selección de reproductores: el efecto promedio (BLUP) por sí solo puede ser insuficiente — machos con buen mérito pero baja consistencia (M1) implican mayor riesgo que machos con mérito moderado y alta consistencia (M3)
